####  模块设计

----

**generate_data.py**

1. 读入需要生成的数据个数n，(默认为2^30)；
2. 生成n个30位的整数，并打印到屏幕。

**sort.cpp**

1. 读入数据个数n；
2. 对数据进行升序排序；
3. 对排序后的数据进行验证；
4. 保存数据 (排序前，排序后)。

#### 使用说明

---

```bash
g++ sort.cpp -o sort
python generate_data.py [100] | ./sort
#注：.py后的[]部分为可选，可以是大于零的整数，若不带有参数，则默认生成数据数目为n=2^30。
```
